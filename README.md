### Quick start on Debian host :
```
sudo apt install lld llvm clang git qemu-system
git config http.postBuffer 524288000
git clone https://github.com/3-o-3/3o3.git
cd 3o3
mkdir bin
cd bin
sh ../build.cmd .. all
cd ..
codium .
```
